# 指令介绍

【指令】罗德岛午夜逸闻 指令表

# 1. 查看简易指令说明 help

#kr help

会输出一段关于指令的文本。简要记载了指令的作用。

# 2. 存档 sav

#kr new                // 新建存档
#kr sav 0[1,2,...]     // 设置默认存档编号(变更当前存档) 
#kr sav show           // 查看当前默认存档

游戏采用群共同存档机制。每个群可以保存多个存档。

使用#kr new 建立存档，使用#kr sav show展示存档名称，如下图所示。

存档不消耗信用，存档命名以群号_编号.json为规则。其中编号即为存档的编号。可以通过指令对存档编号进行修改，改变当前群进行故事的存档。

使用#kr sav 3 表示将默认存档修改为群号_3.json这个0号存档。如下图所示。

# 3. 查看存档状态 show

*展示存档 #kr show

在游戏的任何地方，博士都可以使用#kr show指令来查看当前的存档状态。包括信用池、同伴、道具等等。

# 4. 行动 go

*继续前进 #kr go

当一个操作结束后，凯尔希会保存并等待下一个指令。此时输入#kr go可以继续前进，查看当前文本的要求。#kr go行动不消耗存档信用池的信用。因此每个群成员皆可以使用。

# 5. 信用筹集 crd

哼哼，你以为这个功能全都可以免费使用吗？想得美！你需要支付在凯尔希处签到获得的信用点，才能参与决策行动！

每个群共用同一个信用池，同时，一个新存档也需要大家一起凑够一定的信用值才能开始活动。（数值待定）

否则就会触发下面的对话：

*信用众筹x点 #kr crd x

你可以在任何时候，对当前存档使用这句指令。如#kr crd 50，即向本群的当前默认存档里投入了50信用。

TIP

信用消耗为50，当你的信赖达到200时，实际信用消耗为你输入值的一半。半价！

上图中需要注意的一点是：每个群的每个存档都会设置一份可以参与决策行动列表的名单，暂时称为神经连接者吧。每个人（包括建立存档的人）都不在这个名单中，当且仅当博士**【一次性筹集信用】**超过某个限定值时，你才能让自己加入这份名单。

只有加入名单的人，才可以使用影响故事走向的重要行动指令。（如选择、掷骰子等）

游戏测试阶段与测试者权限者 不消耗信用。暂定一次性投入40，所有人总额超过400点才能启动一场游戏。暂定，暂定……

# 6. 重要行动 ch、d

*进行选择[消耗信用点] #kr ch 0[1,2,...]
*进行掷骰[消耗信用点] #kr d

重要行动需要玩家在神经连接名单中。具体参见3.信用筹集部分。

  1. 在凯尔希的文本中，若出现：

    choice:
0: xxxxxx
1: xxxxxx

    说明此处为选择节点，应当使用#kr ch x进行选择，其中x指的是文本中冒号前面的数字，代表了选项的编号。选择后会自动呈现当前节点的后续文本。结束后可以使用#kr go继续前进。

    选择行动会消耗信用池的信用。消耗值待定。

  2. 在凯尔希的文本中，若出现：

    行动：掷骰 1d100

    说明此处为掷骰节点，应当使用#kr d进行掷骰子，凯尔希会自动帮你掷出相应的骰子，并把结果呈现给你。结束后可以使用#kr go继续前进。

    掷骰行动会消耗信用池的信用。消耗值待定。
